# core-backend

Центральный модуль логики и каталога приложения.

## Назначение модуля
Этот сервис выполняет роль ядра системы. Он отвечает за управление метаданными ИИ-фильмов, профилями пользователей, социальным взаимодействием и предоставляет структурированный API для фронтенда через слой Gateway.

## Основной функционал
* **Управление каталогом:** Хранение и выдача списков видео, разбитых по категориям и жанрам.
* **Энциклопедия контента:** Хранение детальной информации о каждом ролике (использованные промпты, задействованные нейросети вроде Runway, Luma, DeepSeek, авторы генерации).
* **Взаимодействие с БД:** Прямая работа с реляционной базой данных (PostgreSQL) для обеспечения целостности данных приложения.

## Технологический стек
* **Язык разработки:** Golang
* **База данных:** PostgreSQL
